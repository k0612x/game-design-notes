## 🎯 关于这个项目

**Always Have Fun** 是我业余时间独立开发的一款叙事向游戏原型，使用 Godot 引擎。

- 核心概念：后启示录世界中「被社会抛弃的自我意识」如何在蝗灾环境中寻找存在意义
- 文风参考：《银河系搭车指南》式荒诞叙事，融合通感修辞
- 玩法方向：CRPG分支叙事 + 生存资源管理
- 当前阶段：故事线原型 + Godot 技术验证
